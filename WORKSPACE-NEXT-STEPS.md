# Suggested Google Workspace path for CSCA

## Most realistic current path
Because CSCA is operating as a nonprofit organization rather than an accredited school, the more realistic current route is:
1. Apply for **Google for Nonprofits**
2. Activate **Google Workspace for Nonprofits**
3. Use that as your official organization email/collaboration system

## Future path
If CSCA later launches a government-recognized / accredited school or other qualifying educational institution:
- create a **separate school-specific domain**
- apply for **Google Workspace for Education** with that school entity and its accreditation documents

## Public website signals that help
- Official domain
- Legal status
- Governance roles
- Real contact addresses
- Active updates page
